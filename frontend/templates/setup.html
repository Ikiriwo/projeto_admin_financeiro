Configure sua chave API do Google Gemini

Atenção: O sistema precisa de uma chave API do Google Gemini para funcionar.
Primeira vez usando? Não se preocupe! Vamos te guiar passo a passo.
A chave será salva localmente no servidor e não será compartilhada.
Como obter sua chave API:
  1. Acesse Google AI Studio
  2. Faça login com sua conta Google
  3. Clique em "Create API Key"
  4. Copie a chave gerada
  5. Cole acima e clique em "Validar e Salvar"
Segurança: Sua chave é armazenada apenas localmente no arquivo .env do servidor. Este arquivo não é sincronizado com o Git e permanece privado.
